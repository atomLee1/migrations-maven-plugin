migrations:script

Full name:

org.mybatis.maven:migrations-maven-plugin:1.1.1:script

Description:

Goal which executes the ibatis migration script command.

Attributes:

  • Requires a Maven 2.0 project to be executed.

Required Parameters

Name Type Since Description
v2 String - Final version.

Optional Parameters

Name Type Since Description
environment String - Environment to configure. Default environment is 'development'.
Default value is: development.
force boolean - Forces script to continue even if SQL errors are encountered.
Default value is: false.
output File - The output file to be create.
repository File - Location of migrate repository.
Default value is: ..
skip boolean - Skip migration actions.
Default value is: false.
v1 String - Initial version

Parameter Details

environment:

Environment to configure. Default environment is 'development'.
  • Type: java.lang.String
  • Required: No
  • Expression: ${migration.env}
  • Default: development

force:

Forces script to continue even if SQL errors are encountered.
  • Type: boolean
  • Required: No
  • Expression: ${migration.force}
  • Default: false

output:

The output file to be create.
  • Type: java.io.File
  • Required: No
  • Expression: ${migration.output}

repository:

Location of migrate repository.
  • Type: java.io.File
  • Required: No
  • Expression: ${migration.path}
  • Default: .

skip:

Skip migration actions.
  • Type: boolean
  • Required: No
  • Expression: ${migration.skip}
  • Default: false

v1:

Initial version
  • Type: java.lang.String
  • Required: No
  • Expression: ${migration.v1}

v2:

Final version.
  • Type: java.lang.String
  • Required: Yes
  • Expression: ${migration.v2}
